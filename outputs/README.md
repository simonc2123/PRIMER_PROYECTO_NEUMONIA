# Carpeta de Salidas

Almacena todos los archivos generados por GUI y CLI:

- **`heatmaps/`** - Mapas de calor (Grad-CAM) en formato JPG
- **`reports/`** - Reportes PDF generados
- **`historial.csv`** - Registro de todos los análisis realizados

**Nota:** Esta carpeta se regenera automáticamente si se elimina.
